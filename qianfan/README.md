# Qianfan的API接口使用说明
百度的APIkey获取方式有点不一样

## 将Qianfan的api密钥导入到环境变量中
  
> 分linux和windows

### Linux

- 在终端中输入以下命令：
  
    ```bash
    export Qianfan_API_KEY=your_api_key
    ```

### Windows
- 在命令提示符中输入以下命令：
    ```bash
    setx Qianfan_API_KEY "your_api_key_here"
    ```

---

## 安装Qianfan的Python SDK

- 在终端中输入以下命令：

    ```bash
    pip install qianfan
    ```
